# Scaffolding overview

This workspace now contains two minimal, dependency-free scaffolds you can run immediately:

- `node-app/` — Node HTTP server using built-in `http` module (port 3000)
- `python-app/` — Python HTTP server using `http.server` (port 8000)

Quick run commands:

```bash
# Node
node node-app/server.js

# Python
python3 python-app/app.py
```

Use these as templates to expand into a full backend, frontend, or tool script.
